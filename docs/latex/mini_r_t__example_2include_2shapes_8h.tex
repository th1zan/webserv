\mbox{\hyperlink{mini_r_t__example_2include_2shapes_8h_aebe0c6c62ec20bff89b7fdaebd488c75}{t\+\_\+discriminant}} sphere\+\_\+discriminant (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{mini_r_t__example_2include_2shapes_8h_a491fe2df71f2e6a31de7c05d15012f6c}{t\+\_\+sphere}} \texorpdfstring{$\ast$}{*}}]{sphere}{, }\item[{\mbox{\hyperlink{rays_8h_a249ca701a99a7da39d83c6270adac2ec}{t\+\_\+ray}}}]{ray}{}\end{DoxyParamCaption})}



This function returns the necessary data to evaluate a ray-\/sphere intersection. To do that, it relies in geometry, trigonometry and the Pythagorean theorem. It\textquotesingle{}s important to mention that discriminant means the number that says whether the ray intersects the sphere at all. 


\begin{DoxyParams}{Parameters}
{\em sphere} & \doxylink{class_a}{A} pointer to a structure of type {\ttfamily t\+\_\+sphere} representing the sphere to be (or not) intersected with a ray. \\
\hline
{\em ray} & \doxylink{class_a}{A} structure of type {\ttfamily t\+\_\+ray} representing the ray that is going to intersect (or not) the sphere. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
Returns a structure of type {\ttfamily t\+\_\+discriminant} with a, b, c values from quadratic equation derived from initial formulas as well as t1, t2 and discriminant values. t1 and t2 are the distance from ray\textquotesingle{}s origin up to the intersection of given ray with sphere\textquotesingle{}s surface. 
\end{DoxyReturn}

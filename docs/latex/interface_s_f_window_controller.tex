\doxysubsection{Detailed Description}
Implementation of \doxylink{protocol_window_impl_delegate_protocol-p}{Window\+Impl\+Delegate\+Protocol} for window management. 

Key, mouse and Window focus events are delegated to its view, \doxylink{interface_s_f_open_g_l_view}{SFOpen\+GLView}.

Used when SFML handle everything and when a NSWindow\texorpdfstring{$\ast$}{*} is given as handle to Window\+Impl.

When grabbing the cursor, if the window is resizeable, m\+\_\+restore\+Resize is set to YES and the window is marked as not resizeable. This is to prevent accidental resize by the user. When the cursor is released, the window style is restored.

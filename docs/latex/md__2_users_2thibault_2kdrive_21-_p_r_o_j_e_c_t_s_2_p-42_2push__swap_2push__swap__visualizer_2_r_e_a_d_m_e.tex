This project is a visualizer for the {\bfseries{"{}\+PUSH\+\_\+\+SWAP"{}}} 42 School Project.

{\bfseries{Push Swap}} is a program that takes as argument a space separated list of numbers and outputs a list of commands that can be used to sort them.

You can read the subject \href{https://github.com/Binary-Hackers/42_Subjects/blob/master/00_Projects/02_Algorithmic/push_swap.pdf}{\texttt{ here}}.\hypertarget{md__2_users_2thibault_2kdrive_21-_p_r_o_j_e_c_t_s_2_p-42_2push__swap_2push__swap__visualizer_2_r_e_a_d_m_e_autotoc_md881}{}\doxysection{\texorpdfstring{Usage}{Usage}}\label{md__2_users_2thibault_2kdrive_21-_p_r_o_j_e_c_t_s_2_p-42_2push__swap_2push__swap__visualizer_2_r_e_a_d_m_e_autotoc_md881}

\begin{DoxyItemize}
\item If you add the program inside your project, you will need to add {\bfseries{../../}} before the name of your program to make the visualizer work.
\item In the {\bfseries{Values}} window
\begin{DoxyItemize}
\item Choose the size of the push swap input with the slider \mbox{[}Optional\mbox{]}
\item {\bfseries{Shuffle}} the input \mbox{[}Optional\mbox{]}
\item The space separated values should be filled automatically, you can also put your own values
\item Set the Push Swap program path (Absolute or relative path).
\item {\bfseries{Compute}} the sort commands, it will display OK when done.
\end{DoxyItemize}
\item In the {\bfseries{Controls}} window
\begin{DoxyItemize}
\item {\bfseries{Load}} the commands in the visualizer
\item {\bfseries{Start}} the animation
\item Adjust the {\bfseries{Speed}} as needed
\item {\bfseries{Pause}} and go {\bfseries{Step}} by step to see the details of your algorithm.
\item {\bfseries{Load}} to restart the animation
\end{DoxyItemize}

\doxysubsection{Detailed Description}
The type t\+\_\+pattern contatins all the necessary information to apply a texture on a given shape\textquotesingle{}s surface.

\hline
{\em pattern\+\_\+at} & Receives a pattern function to apply the corresponding pattern. For more information, see t\+\_\+pttr\+\_\+at description. \\
\hline
{\em texture\+\_\+map} & This structure creates a t\+\_\+pattern subclass (the same as t\+\_\+pttr\+\_\+at, which calls \doxylink{patterns_8h_a7bcf02fbe5ca0ab70696e76dbbb681ed}{stripe\+\_\+at()} or \doxylink{patterns_8h_a2fae2c8413c2e3b9874c48196fa44509}{checkered\+\_\+at()}). t\+\_\+text\+\_\+map is a subclass that uses \doxylink{patterns_8h_a64b3488349a691f86c9b541b1d5da0a2}{spherical\+\_\+map()}, \doxylink{patterns_8h_aa7a410c6d9850b0f05939a77f25b5e5d}{planar\+\_\+map()} or \doxylink{patterns_8h_ab3c393fe016f7b74e6d834d655e7bb27}{cylindrical\+\_\+map()} to define how to apply a pattern on a given 2D or 3D shape\textquotesingle{}s surface. \\
\hline
{\em width} & Receives u, the horizontal value of checkers squares. \\
\hline

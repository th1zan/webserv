This function get the inverse of a given matrix. That is, if you multiply some matrix \doxylink{class_a}{A} by another matrix \doxylink{class_b}{B}, producing \doxylink{class_c}{C}, you can multiply \doxylink{class_c}{C} by the inverse of \doxylink{class_b}{B} to get \doxylink{class_a}{A} again. To invert the matrix, this function employs a method known as cofactor expansion. Additionally, there some interesting properties of the inverse\+: 1) if you invert an identity matrix, the result is also an identity matrix. 2) when you multiply a matrix by its inverse, you get as result the identity matrix. 3) The inverse of the transpose of a matrix and the transpose of the inverse only commute if the matrix is symmetric or orthogonal. Otherwise, the result of the two operations will always be different.

\end{DoxyParams}

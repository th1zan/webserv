\doxysubsection{Detailed Description}
Represents a scanner. 

The scanner structure is used for scanning and tokenizing input text.

The scanning process is performed using two pointers, {\ttfamily consume} and {\ttfamily current}, which need to be kept separate. While {\ttfamily current} reads from the input and checks if the character sequence is correct, {\ttfamily consume} remains pointing to the beginning of the portion being verified. This eliminates the need to split the input into pieces and allows for precise identification of the section of input that needs to be converted.

If you have SFML installed on your system, you don\textquotesingle{}t need to set SFML\+\_\+\+DIR during configuration.

You can also specify {\ttfamily BUILD\+\_\+\+SHARED\+\_\+\+LIBS=ON} to build Im\+Gui-\/\+SFML as a shared library. To build Im\+Gui-\/\+SFML examples, set {\ttfamily IMGUI\+\_\+\+SFML\+\_\+\+BUILD\+\_\+\+EXAMPLES=ON}. To build imgui-\/demo.\+cpp (to be able to use {\ttfamily Im\+Gui\+::\+Show\+Demo\+Window}), set {\ttfamily IMGUI\+\_\+\+SFML\+\_\+\+IMGUI\+\_\+\+DEMO=ON}.

After the building, you can install the library on your system by running\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{cmake\ -\/-\/build\ .\ -\/-\/target\ install}

\end{DoxyCode}


If you set {\ttfamily CMAKE\+\_\+\+INSTALL\+\_\+\+PREFIX} during configuration, you can install Im\+Gui-\/\+SFML locally.

Integrating into your project is simple.

\begin{DoxyItemize}
\item Call {\ttfamily Im\+Gui\+::\+SFML\+::\+Init} and pass your {\ttfamily \doxylink{classsf_1_1_window}{sf\+::\+Window}} + {\ttfamily \doxylink{classsf_1_1_render_target}{sf\+::\+Render\+Target}} or {\ttfamily \doxylink{classsf_1_1_render_window}{sf\+::\+Render\+Window}} there. You can create your font atlas and pass the pointer in Init too, otherwise the default internal font atlas will be created for you.
\item For each iteration of a game loop\+:
\begin{DoxyItemize}
\item Poll and process events\+:

{\ttfamily cpp \doxylink{classsf_1_1_event}{sf\+::\+Event} event; while (window.\+poll\+Event(event)) \{ Im\+Gui\+::\+SFML\+::\+Process\+Event(event); ... \} }
\item Call {\ttfamily Im\+Gui\+::\+SFML\+::\+Update(window, delta\+Time)} where {\ttfamily delta\+Time} is {\ttfamily \doxylink{classsf_1_1_time}{sf\+::\+Time}}. You can also pass mouse\+Position and display\+Size yourself instead of passing the window.
\item Call Im\+Gui functions ({\ttfamily Im\+Gui\+::\+Begin()}, {\ttfamily Im\+Gui\+::\+Button()}, etc.)
\item Call {\ttfamily Im\+Gui\+::\+End\+Frame} after the last {\ttfamily Im\+Gui\+::\+End} in your update function, if you update more than once before rendering. (e.\+g. fixed delta game loops)
\item Call {\ttfamily Im\+Gui\+::\+SFML\+::\+Render(window)}
\end{DoxyItemize}
\item Call {\ttfamily Im\+Gui\+::\+SFML\+::\+Shutdown()} after {\ttfamily window.\+close()} has been called
\end{DoxyItemize}

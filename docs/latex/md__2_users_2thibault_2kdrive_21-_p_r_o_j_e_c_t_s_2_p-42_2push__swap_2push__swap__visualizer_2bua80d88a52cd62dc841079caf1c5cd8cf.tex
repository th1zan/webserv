Catch is great for open source. With its \href{../LICENSE.txt}{\texttt{ liberal license}} and single-\/header, dependency-\/free, distribution it\textquotesingle{}s easy to just drop the header into your project and start writing tests -\/ what\textquotesingle{}s not to like?

As a result Catch is now being used in many Open Source projects, including some quite well known ones. This page is an attempt to track those projects. Obviously it can never be complete. This effort largely relies on the maintainers of the projects themselves updating this page and submitting a PR (or, if you prefer contact one of the maintainers of Catch directly, use the \href{https://groups.google.com/forum/?fromgroups\#!forum/catch-forum}{\texttt{ forums}}), or raise an \href{https://github.com/philsquared/Catch/issues}{\texttt{ issue}} to let us know). Of course users of those projects might want to update this page too. That\textquotesingle{}s fine -\/ as long you\textquotesingle{}re confident the project maintainers won\textquotesingle{}t mind. If you\textquotesingle{}re an Open Source project maintainer and see your project listed here but would rather it wasn\textquotesingle{}t -\/ just let us know via any of the previously mentioned means -\/ although I\textquotesingle{}m sure there won\textquotesingle{}t be many who feel that way.

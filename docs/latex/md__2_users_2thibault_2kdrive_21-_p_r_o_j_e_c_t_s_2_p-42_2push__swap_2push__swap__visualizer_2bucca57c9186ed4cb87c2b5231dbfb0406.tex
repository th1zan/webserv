Catch2 is a multi-\/paradigm test framework for C++. which also supports Objective-\/C (and maybe \doxylink{class_c}{C}). It is primarily distributed as a single header file, although certain extensions may require additional headers.\hypertarget{md__2_users_2thibault_2kdrive_21-_p_r_o_j_e_c_t_s_2_p-42_2push__swap_2push__swap__visualizer_2bucca57c9186ed4cb87c2b5231dbfb0406_autotoc_md643}{}\doxysubsection{\texorpdfstring{How to use it}{How to use it}}\label{md__2_users_2thibault_2kdrive_21-_p_r_o_j_e_c_t_s_2_p-42_2push__swap_2push__swap__visualizer_2bucca57c9186ed4cb87c2b5231dbfb0406_autotoc_md643}

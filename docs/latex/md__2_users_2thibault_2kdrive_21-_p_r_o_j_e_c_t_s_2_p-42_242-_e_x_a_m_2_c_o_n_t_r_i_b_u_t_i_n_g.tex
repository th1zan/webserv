You can add your own exercises by simply creating a folder for the exercise in the {\ttfamily .subjects} folder.

üëâ 42\+\_\+\+EXAM automatically detects the exercises available in the folders, so you don\textquotesingle{}t have to change anything in the program.

To add an exercise you have to create a folder with the name of the exercise in its respective level (browse the .subjects folder to understand)

\DoxyHorRuler{0}


üîñ For an exercise to be complete, you must add\+:


\begin{DoxyItemize}
\item \doxylink{class_a}{A} subject named {\ttfamily subject.\+en.\+txt} in a folder {\ttfamily attachment} (you can add other files that will be given to the student during the exercise)
\item The exercise solution (example \+: fprime.\+c) ‚ö†Ô∏è WARNING\+: The exercise must be perfect because it will be used by 42\+\_\+\+EXAM to know if the rendering is correct.
\item The correction file {\ttfamily tester.\+sh} which is the file that allows you to send the inputs to test what you want.
\end{DoxyItemize}

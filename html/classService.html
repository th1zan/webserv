Note
The addrinfo structure is used for addressing information in socket programming. It contains various fields for managing socket connections, including options, address family, socket type, and more.

struct addrinfo { int ai_flags; // options int ai_family; // adress family (ipv4, ipv6) (AF_INET, AF_INET6, etc.) int ai_socktype; // kind of socket (SOCK_STREAM, SOCK_DGRAM, etc.) int ai_protocol; // protocol (IPPROTO_TCP, IPPROTO_UDP, etc.) size_t ai_addrlen; // adress size struct sockaddr *ai_addr; // socket addr char *ai_canonname; // canonical nam struct addrinfo *ai_next; // pointer to the nex struct };

The Service class is responsible for initializing services, managing connections, and handling client requests.

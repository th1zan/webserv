\doxysubsection{Detailed Description}
Represents a client connection handling HTTP requests. 

The \mbox{\hyperlink{classClient}{Client}} class is responsible for managing client requests, checking their validity, and interacting with the associated \mbox{\hyperlink{classServer}{Server}} instance.
